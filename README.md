# Cash-tracker

Je to aplikace dělaná pomocí technologie .Net MAUI.
Použité NuGet balíčky: LiveChartsCore.SkiaSharpView.Maui, sqlite-net-pcl, Syncfusion.Maui.Toolkit
Je to aplikace na zařízení Android, i přesto že jde i na další operační systémy. Hlavní funkcialitou je hlídání "cash flow" uživatele. Uživatel má možnost přidávát vlastní kategorie pod kterými budou ukádané položky evidovány.
